# **App Name**: Vantage Vault

## Core Features:

- Firebase Identity Bridge: Secure authentication integration validating email and password credentials via Firebase Auth.
- Elastic UI Layout: A fluid, mobile-first login interface that optimizes itself for desktop, tablet, and mobile breakpoints.
- Adaptive Password Tool: An AI security assistant that provides real-time tool-based strength feedback and security advice as the user types.
- Dynamic Field Validation: Real-time client-side error handling for email formats and password requirements.
- State Persistence: Managed authentication states using Next.js route protection and session handling.
- Smart Recovery Flow: A minimalist interface for triggering password reset emails directly through the Firebase gateway.

## Style Guidelines:

- Primary color: Electric Indigo (#8D82FA) for a modern, high-tech security vibe.
- Background color: Deep Onyx (#0E0D10), creating a sophisticated dark-mode atmosphere.
- Accent color: Cobalt Flux (#1449B8), used for interactive highlights and high-contrast call-to-action states.
- A combination of 'Space Grotesk' for high-impact headlines and 'Inter' for clean, readable form inputs and button text.
- Centered glassmorphic container for the login card to provide visual focus against a desaturated backdrop.
- Sleek, minimalist outline icons for field indicators and visibility toggles.
- Micro-interactions on input focus and a subtle lateral shift transition during validation errors.
